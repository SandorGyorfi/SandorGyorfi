# Hi, I'm Sandor

I design, build, and ship full products — from idea to production.

I work across the entire stack: web, desktop, real-time systems, and AI-powered applications. I don't just write features — I architect platforms, set up infrastructure, and take products from zero to launch.

## What I Build

- **SaaS platforms** with complex business logic, multi-tenant architecture, and payment systems
- **Desktop applications** using Tauri and Rust
- **AI-integrated tools** with OCR, language models, and intelligent automation
- **Real-time systems** with WebSockets, task queues, and event-driven backends
- **Internationalized apps** serving multiple markets and languages
